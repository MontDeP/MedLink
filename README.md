# Projeto - Sistema de Agendamento MedLink

**O MedLink é um sistema de agendamento de consultas desenvolvido como parte da disciplina de Projeto de Sistemas.**  

O projeto tem como objetivo oferecer uma solução **simples, acessível e eficiente** para clínicas conjuntas de médicos, geralmente profissionais em início de carreira que precisam de mais organização no dia a dia.  

O sistema permite que secretárias e médicos façam login, gerenciem suas agendas, cadastrem pacientes e reduzam perdas de produtividade causadas por consultas esquecidas ou não comparecimentos.

## User Stories

### US001 - Autenticação de Usuários
**Como** Secretária/Médico, **quero** me autenticar com CPF/senha **para** acessar meu perfil.  
**Critérios de Aceite:** Login deve validar CPF e senha com segurança.

### US002 - Recuperação de Senha
**Como** Usuário, **quero** redefinir minha senha via e-mail **para** recuperar o acesso.  
**Critérios de Aceite:** Envio de link seguro de redefinição.

### US003 - Cadastro de Paciente
**Como** Secretária, **quero** cadastrar um novo paciente **para** registrar seus dados no sistema.  
**Critérios de Aceite:** Inclusão de nome, CPF, telefone e e-mail obrigatórios.

### US004 - Agendamento de Consulta
**Como** Secretária, **quero** agendar uma consulta validando conflitos de horário **para** evitar sobreposição.  
**Critérios de Aceite:** Sistema deve impedir agendamentos conflitantes.

## 📝 Valor a ser Entregue - 1ª Sprint

### Resumo:
- ⬜ Implementação do sistema de login com CPF/senha.  
- ⬜ Função de recuperação de senha via e-mail.  
- ⬜ Módulo de cadastro de pacientes.  
- ⬜ Funcionalidade de agendamento com validação de conflitos.
